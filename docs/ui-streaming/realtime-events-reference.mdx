---
title: Realtime & Events Reference
description: Event schema consumed by the UI reducer.
---

Core events:

- Lifecycle: `run.started`, `run.completed`, `run.failed`, `run.interrupted`, `stream.ended`
- Parts: `part.created`, `part.completed`
- Deltas: `text.delta`, `tool_call.arguments.delta`, `tool_call.output.delta`
- Other: `usage.updated`, `metadata.updated`, `hitl.requested`/`hitl.resolved` (TBA UI)

Ordering:

- Each event includes `sequenceNumber` for monotonic ordering
